MobileKeysApduCommand

@interface MobileKeysApduCommand : NSObject

The APDU Command Superclass. Use this class either to construct APDU Commands from raw APDU data, or subclass it. This class WILL allow APDUs with larger payloads than 255 bytes

See

APDU format overview on Wikipedia

Note

since version 5.0.0
