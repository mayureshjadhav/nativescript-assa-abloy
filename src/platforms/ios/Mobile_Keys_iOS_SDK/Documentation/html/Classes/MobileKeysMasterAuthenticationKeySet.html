MobileKeysMasterAuthenticationKeySet

@interface MobileKeysMasterAuthenticationKeySet : MobileKeysAuthenticationKeySet

Represents a master authentication credential, used to calculate derived keys based on diversification data.

See

Section 2.1 in the Seos Core Technology Specification

    Swift

    init!(keyNumber: MobileKeysKeyNumber, masterKeyData masterKey: Data!, oid: Data!, diversifier: Data!, encryptionAlgorithm: MobileKeysEncryptionAlgorithm, global: Bool)

    Parameters

    keyNumber

    the key number this key should authenticate to.
    global

    true of the referred key is global, false if the key number refers to a ADF local key.
    masterKey

    the master key data, 16 bytes AES key used to derive card specific keys.
    oid

    oid to use during key derivation, selected Oid used if null
    diversifier

    diversifier to use during key derivation, selected ADF Diversifier used if null
