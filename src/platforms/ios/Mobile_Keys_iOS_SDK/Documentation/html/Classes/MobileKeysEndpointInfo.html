MobileKeysEndpointInfo

@interface MobileKeysEndpointInfo : NSObject

Class representing Information about a Seos Endpoint. Most of this information is Seos specific and rather technical. In addition to what gets read from Seos, this class also contains the current Push ID (as set through the Mobile Keys SDK) and a timestamp (NSDate) describing when the endpoint was last successfully synchronized with the Seos TSM.
